# mongo-go

## Description

API rest with Golang, MongoDb and authentication

## Getting Started

### Dependencies

* go 1.19
* mongodb database available

### Installing

* clone the repository
* set the environment variables

### Executing program

* go build
* go mod tidy
* go run main.go
